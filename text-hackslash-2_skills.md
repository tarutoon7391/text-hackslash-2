# text-hackslash-2 スキル表（実装用データ仕様）

最終更新：2026-07-22
GDD 9番「スキルシステム」の付録。skills_normal.js / skills_elemental.js のデータソース。

## 共通ルール

- 多段技は「1ヒット倍率×ヒット数」。付与・会心判定は各ヒット独立
- 状態異常付与率：低=20%／中=35%／高=50%
- バフ/デバフは3ターン持続が基本（明記があればそちら優先）
- バフ/デバフ：同効果重ね掛け不可、再使用で残りターン上書き
- 「弱体効果」=状態異常＋デバフの総称。参照系は両方カウント、解除系は両方対象
- 会心は2.0倍（「会心時ダメージX倍」表記のスキルはその値で上書き）
